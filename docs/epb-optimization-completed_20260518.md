## 二、新增核心模块

### 1. 智能案件分析模块（smart_analyzer.py）

**功能：**
- 案情信息自动提取（当事人、污染物、超标倍数、金额、法条）
- 违法类型智能识别（8大类、50+关键词匹配）
- 法条智能推荐（违法行为→法条→处罚标准）
- 风险评估（评分制，高/中/低风险分级）
- 证据清单自动生成
- 相似案例智能匹配

**测试结果：**
```
输入："某化工厂私设暗管排放含重金属废水，超标5倍"

输出：
- 违法类型：水污染类
- 风险等级：高风险（评分110）
- 适用法条：《水污染防治法》第83条、《刑法》第338条
- 涉刑判断：是（超标3倍以上或含重金属）
- 证据清单：7项
- 相似案例：3个
```

### 2. 法条关联数据库（law_mapping.json）

**覆盖违法类型：**
- 水污染类（私设暗管、超标排放、设施不正常运行）
- 大气污染类（超标排放、VOCs无组织排放、篡改监测数据）
- 固体废物类（非法倾倒危废、台账不规范）
- 环评类（未批先建、未验先投）
- 排污许可类（无证排污、超许可排污）
- 自动监控类（在线监测不正常运行、篡改监测数据）
- 噪声污染类（工业噪声超标）
- 移动源监管类（柴油车违规、加油站、非道路机械、检测机构）

**包含内容：**
- 违反条款
- 处罚依据
- 刑事门槛判断
- 裁量基准参考
- 证据要求清单

- 风险等级评估

## 四、技术实现亮点

### 1. 智能提取算法

使用正则表达式从案情描述中自动提取：
- 当事人名称（公司、厂、企业等）
- 污染物类型（COD、氨氮、重金属、VOCs等）
- 超标倍数
- 罚款金额
- 提及的法条

### 2. 风险评估算法

```
风险评分 = 
  污染物权重（重金属+30，危废+40，有毒+50）
  + 超标倍数权重（≥10倍+30，≥3倍+20，≥1倍+10）
  + 涉刑判断（+40）
  + 高风险关键词（篡改、暗管等+20）

高风险：≥70分
中风险：40-69分
低风险：<40分
```

### 3. 法条推荐逻辑

```
案情描述 → 关键词匹配 → 违法类型识别 → 法条映射表查询 → 返回推荐法条
```
