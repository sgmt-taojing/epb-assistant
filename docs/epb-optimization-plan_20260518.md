### 4.2 风险评估算法
```python
def calculate_risk(case):
    score = 0
    # 污染物类型权重
    if case.get('heavy_metal'): score += 30
    if case.get('hazardous_waste'): score += 40
    if case.get('toxic'): score += 50
    
    # 超标倍数权重
    excess_ratio = case.get('excess_ratio', 1)
    if excess_ratio > 3: score += 20
    if excess_ratio > 10: score += 30
    
    # 敏感区域
    if case.get('sensitive_area'): score += 20
    
    # 是否涉刑
    if case.get('criminal'): score += 50
    
    return "高风险" if score >= 70 else "中风险" if score >= 40 else "低风险"
```
