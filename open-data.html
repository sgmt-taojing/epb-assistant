接入指南
快速接入
1. 所有API均为RESTful风格，支持JSON格式
2. GitHub Pages静态数据可直接访问（无需认证）
3. 本地/Tunnel服务支持完整读写API
4. CORS已开启，支持跨域调用
5. 无需API Key，开源协议MIT
